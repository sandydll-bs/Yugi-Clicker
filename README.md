# Yugi Clicker

Yugi Clicker is an autoclicker designed to operate discreetly while providing basic protection against remote monitoring environments.

## Features

### Streamproof
- Designed to remain hidden during screen sharing, streaming, and recording sessions.

### AnyDesk Protection
- The application will not start if AnyDesk is detected running on the system.
